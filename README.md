# native-php-sistem-informasi-antrian
Sistem Informasi Antrian Menggunakan PHP Murni. <br>
Dapat dikembangan menggunakan Framework Yang disukai.

Fitur :
- Antrian Beras Bulog
- Riwayat Pengambilan
- Laporan Antrian Beras Bulog
- Laporan Pengambilan
- Cetak Laporan
