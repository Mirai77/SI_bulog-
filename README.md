# native-php-sistem-informasi-arsip-pengadilan
Sistem Informasi Arsip Pengadilan Menggunakan PHP Murni. <br>
Dapat dikembangan menggunakan Framework Yang disukai.

Fitur :
- Arsip Perkara 
- Riwayat Peminjaman Arsip
- Laporan Arsip Perkara
- Laporan Peminjaman
- Cetak Laporan
